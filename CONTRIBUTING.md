# Contributing to Timeweb SDK

Thank you for your interest in contributing to Timeweb SDK! Here are some basic guidelines:

## How to Submit an Issue
1. Before creating a new issue, please check if a similar one already exists.
2. Fill out the appropriate issue template (Bug Report / Feature Request).

## How to Submit a Pull Request
1. Fork the repository.
2. Create a new branch (`feature/my-new-feature`).
3. Make your changes and ensure all tests pass.
4. Submit a PR using the provided template.

## Coding Guidelines
- Follow PEP 8 for Python code style.
- Keep changes modular and well-documented.
